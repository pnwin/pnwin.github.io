DIY Photoshop

DIY Photoshop was a project in which I was able to code (with p5js) an entire tool that is dedicated, and created for one specific purpose of my choice: To construct the best digital computer setup ever. This tool features hotkeys that provide respective funtions such as placing a desk mat, pc, and so on. Along with a variety of peripherals like wired/wireless mice, big/small computer cases, the limit is truly endless. With each element carefully designed and hand-drawn with care, be prepared to create the greatest setup ever! The code is also readily available to explore!
